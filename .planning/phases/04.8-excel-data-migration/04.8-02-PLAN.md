<objective>
**Frontend Ingestion UI**

This plan delivers the user interface to drive the Excel data migration. It builds a hidden route (`/migration`), a drag-and-drop file uploader, a safety-critical confirmation modal containing a destructive wipe warning, a text challenge requiring the user to type "MIGRATE" to confirm, and a polling-based progress state machine that displays real-time sheet ingestion metrics before redirecting to the main dashboard upon success.

Key Outputs:
- Exposing `/migration` via React Router, keeping it hidden from navigation.
- A drag-and-drop zone with full dragover states and file type constraints (accepting only `.xlsx`).
- A confirmation modal warning of cascade DB wipes and enforcing the challenge.
- A polling system querying the job status every 2000ms.
- A visually appealing progress bar using Tailwind CSS v4 gradients.
- Success and failed transition states matching the design system specifications.
</objective>
