- **No `baseURL` override:** The Better Auth client defaults to `/api/auth` which matches the Vite proxy forwarding `/api/auth/*` to `http://localhost:3000`. `credentials: 'include'` is set automatically.
- **Google button always visible:** The Google OAuth button remains visible regardless of whether Google credentials are configured — errors are handled gracefully inline.
- **Sign-up auto-sign-in:** Better Auth automatically signs the user in after successful sign-up (default behavior), so no additional sign-in call is needed after registration.
