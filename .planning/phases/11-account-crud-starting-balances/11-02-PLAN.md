| Boundary | Description |

| UI → user action | Typed delete confirmation prevents accidental deletion — challenge phrase must exactly match "DELETE {name}" |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-05 | Information Disclosure | AccountPage data | mitigated | Existing Phase 10 per-user key scoping: queryKeys.accounts.all(userId) prefix with ['user', userId] — cross-user data cannot display |
| T-11-06 | Repudiation | Delete confirmation | mitigated | TypedDeleteConfirmModal requires user to type "DELETE {name}" — no single-click delete; case-sensitive match prevents accidental confirms |
