## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, card gaps |
| xl | 32px | Container padding |
| 2xl | 48px | Major section breaks, page-level padding |
| 3xl | 64px | Page-level spacing |

Exceptions: Icon-only touch targets use 44px minimum (for insight dismiss buttons, tab icons).

**Sources:** Extracted from existing components — `p-6` on DashboardPage cards (24px), `p-8` on ImportUpload container (32px), `space-y-6`/`gap-6` for grid layouts (24px), `py-12` in App.tsx main content (48px). Standard Tailwind spacing scale. [Codebase analysis]

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#020617` (slate-950) | Page background, body |
| Secondary (30%) | `#0f172a` (slate-900) | Cards (`bg-slate-900/80`), nav header (`bg-slate-950/60`), tooltips |
| Accent (10%) | `#2563eb` → `#4f46e5` (blue-600 → indigo-600 gradient) | Primary CTAs, active nav items, brand logo, AI forecast chart line |
| Destructive | `#450a0a` / `#991b1b` (red-950 / red-800) | Error states (`bg-red-950/50 border-red-800`), destructive buttons |

Accent reserved for:
- Primary CTA buttons (`bg-gradient-to-r from-blue-600 to-indigo-600`)
- Active navigation items (`text-blue-400`, `bg-slate-900`)
- Brand logo (FinanceFlow gradient: `from-blue-600 to-indigo-600`)
- Dashboard AI forecast chart line (`stroke="#06b6d4"` cyan-500, distinct from LR amber)
- Insight priority indicator icons (red/yellow/blue per D-08)
- Page-level gradient headings (optional, following ImportUpload pattern)

**Not accent:** Card borders are `slate-800` (secondary palette). Body text is `slate-100` to `slate-400`. Chart colors follow Recharts defaults.

**Sources:** CTA, empty states, error, and dismiss confirmation — user-approved defaults. Tab names from D-07. ## Chart Contract: ComboChart Extension

The existing ComboChart (`frontend/src/charts/ComboChart.tsx`) must be extended with a second prediction line:

| Property | Existing LR Line | New AI Forecast Line |
|----------|-----------------|---------------------|
| `dataKey` | `prediction` | `aiForecast` |
| `stroke` | `#f59e0b` (amber-500) | `#06b6d4` (cyan-500) |
| `strokeWidth` | 2 | 2 |
| `strokeDasharray` | `"8 4"` (long dashes) | `"4 4"` (short dashes) |
| `connectNulls` | true | true |
| `name` (legend) | Predykcja (LR) | Predykcja (AI) |

**Merged data interface** (extends existing `ComboDataPoint`):
```typescript
interface AugmentedComboDataPoint extends ComboDataPoint {
  prediction: number | null;    // existing LR
  aiForecast: number | null;    // new AI forecast
}
```

## Interaction States

### Insight Card States
| State | Visual |
|-------|--------|
| Default | Full card: icon + title + first 2 lines of content + relative timestamp + "See related transactions" link |
| Dismissed | Card removed from list with slide-out animation (200ms ease-out). Dismissed rows excluded from GET queries. |
| Generating (on-demand) | Loading spinner card: "Analyzing your finances..." with animated spinner |
| Empty (no data) | Centered empty state card with heading + body + CTA button |

### Dashboard Widget States
| State | Visual |
|-------|--------|
| Has insights | 3 compact cards in horizontal row: priority icon (colored circle) + first line of text + relative timestamp |
| No insights yet | Single compact card: "No insights yet. Import transactions to get started." |
| Loading | Skeleton cards (3 shimmer placeholders with `animate-pulse`) |

### Generate Button States
| State | Visual |
|-------|--------|
| Idle | Gradient CTA: `bg-gradient-to-r from-blue-600 to-indigo-600` |
| Hover | Lighter gradient: `hover:from-blue-500 hover:to-indigo-500` |
| Generating | Disabled + spinner: `cursor-not-allowed opacity-60` with animated spinner icon |
| Cooldown | Disabled with tooltip: "Please wait 5 minutes between generation requests" |
| Error | Button reactivates; error card shown above |

### Insight Tabs
| Tab | Icon (inline SVG) | Filter |
|-----|-------------------|--------|
| Alerts | Exclamation triangle | `type = 'alert'` |
| Trends | Trending-up arrow | `type = 'trend'` |
| Tips | Lightbulb | `type = 'tip'` |
| Forecasts | Chart-line | `type = 'forecast'` |

Tabs follow the existing App.tsx nav button pattern: `px-4 py-2 rounded-lg text-sm font-semibold` with active state (`bg-slate-900 text-blue-400`) and inactive state (`text-slate-400 hover:text-slate-200`).

**Sources:** D-07 (type tabs with icons), D-08 (dashboard widget with priority icons). Tab states derived from App.tsx nav pattern. Cooldown from RESEARCH.md rate limit mitigation. [CONTEXT.md + RESEARCH.md + Codebase analysis]
