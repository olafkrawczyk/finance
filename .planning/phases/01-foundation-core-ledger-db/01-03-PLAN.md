<task type="auto">
  <name>Task 1: Implement Post Transaction Logic</name>
  <files>src/core/ledger/use-cases.ts, src/interface-adapters/api/ledger.ts</files>
  <action>
    In `src/core/ledger/use-cases.ts`, implement `postTransaction`:
    - Wraps the creation of a transaction header and its line items in a single `sql.begin` block.
    - Uses the database's atomic commit to ensure the zero-sum trigger runs.
    In `src/interface-adapters/api/ledger.ts`, create a `POST /ledger/transactions` route:
    - Validates input using `TransactionSchema`.
    - Calls `postTransaction`.
    - Returns 201 on success, 400 on validation failure, or 409 if the zero-sum constraint fails.
  </action>
  <verify>
    <automated>curl -X POST -H "Content-Type: application/json" -d '{"description": "Test", "entries": [{"account_id": "A", "amount": "100"}, {"account_id": "B", "amount": "-100"}]}' http://localhost:3000/ledger/transactions</automated>
  </verify>
  <done>Balanced transactions can be posted via API.</done>
</task>
