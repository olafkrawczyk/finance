## Trust Boundaries

| Boundary | Description |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-10 | Information Disclosure | All chart components | accept | Charts render data from props only. Recharts renders SVG elements — no XSS vector. No user-provided data is interpolated as HTML. |
| T-03-11 | Spoofing | onMonthClick drill-down | accept | Click handler navigates to `/month/${activeLabel}` where `activeLabel` is a data-internal month string ("YYYY-MM"). The page component parsing expects this format. Malformed data could produce a 404, not a security issue. |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No new packages installed in this plan. Recharts already installed in Plan 02. |
</threat_model>
