  <action>
    Create `frontend/src/lib/linearRegression.ts`. This is a pure utility module with no imports, no React, no side effects.

    Create the directory `frontend/src/lib/` if it doesn't exist.

    Export two functions:

    **linearRegression(points: Point[]): `{ slope: number; intercept: number; r2: number }`**
    - Point interface: `{ x: number; y: number }`
    - Standard OLS formula: β = (XᵀX)⁻¹Xᵀy
    - Uses simple sum accumulators (sumX, sumY, sumXY, sumX2, sumY2) in a single pass
    - slope = (n * sumXY - sumX * sumY) / (n * sumX2 - sumX * sumX)
    - intercept = (sumY - slope * sumX) / n
    - Computes R² = 1 - SSres / SStot
    - Returns `{ slope: 0, intercept: 0, r2: 0 }` if n < 2

    **predictPoints(historicalData, monthsToPredict): `{ monthIndex: number; value: number }[]`**
    - historicalData: `{ monthIndex: number; value: number }[]`
    - Calls linearRegression() on the data
    - Generates `monthsToPredict` future points starting from `lastIndex + 1`
    - Each prediction: value = slope * monthIndex + intercept

    Use the EXACT implementation from RESEARCH.md Code Examples section (the 50-line version with interface Point, linearRegression function with 6 sum variables, and predictPoints function). No external libraries.

    Add descriptive JSDoc comments at the top:
    ```
    /**
     * Simple OLS linear regression — no external library required.
     * Computes β₁ (slope) and β₀ (intercept) for y = β₀ + β₁x.
     * Uses all historical data points for prediction per D-02.
     */
    ```
