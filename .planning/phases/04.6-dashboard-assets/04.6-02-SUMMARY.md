## Decisions Made

- **DashboardPage receives `onAssetsClick` prop** from App.tsx to enable navigation from the Total Net Value tile to the assets management page
- **Inline editing per row** is used for asset editing (no modal) — faster UX, consistent with modern design patterns
- **Current month fallback** silently selects `data[data.length - 1]` when current month has no data — the month label clearly identifies which month is displayed
- **Y-axis labels formatted as whole numbers** (`maximumFractionDigits: 0`) to keep chart axis clean, while tooltips show full 2-decimal precision
- **SavingsLogChart uses `toFixed(2)` instead of `pl-PL`** because log values are small floating-point indices (e.g., 3.45) where thousands separators are not meaningful
